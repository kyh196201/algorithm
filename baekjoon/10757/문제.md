### 문제

- [큰수 A + B](https://www.acmicpc.net/problem/10757)

### 풀이

```markdown

1. 자바스크립트에서 표현할 수 있는 숫자에는 한계가 있기 때문에, 큰 숫자의 경우 문자열로 입력을 받고 덧셈을 해야한다.

2. A, B가 0 < A, B < 10의 10000승의 범위이다.

3. Number 자료형으로 표현할 경우 10의 309승만 되어도 Infinity라고 표현된다.

4. 문자열의 덧셈?

5. A, B를 split을 통해 배열로 만든 후, reverse를 통해서 역순으로 뒤집는다.

6. 이렇게되면, 1의 자리부터 0번째 인덱스에 놓이게된다.

7. 두 배열 중에 길이가 긴 배열을 기준으로 반복문을 실행한다.

8. 각 자릿수를 더하는데, 더한 값이 10보다 커질 경우 1의 자리만 남기고 10을 suffix에 값을 저장한다.

9. 각 자릿수를 더할때 suffix가 있으면 이 값을 더해주고, 다시 자릿수가 발생하면 값을 할당한다.

10. 길이가 짧은 배열의 경우 인덱스에 해당하는 값이 없을 경우 0으로 변환한다.
```
