### 풀이

```markdown
- 해시 테이블을 이용한 풀이

1. completion 배열을 기준으로 해시 테이블을 생성한다.
2. 참여자 중에 동명 이인이 있을 수 있으므로, 테이블에 이미 해당 name에 대한 property가 존재하는 경우 값에 + 1을 하고, 그렇지 않으면 해당 name에 대한 property를 생성하고 값을 0으로 설정해준다.

3. 이렇게 completion 배열을 모두 순회하면 해시 테이블 안에는 각 name에 대한 등장 횟수가 누적되게된다.

4. 이 테이블을 기준으로 participant 배열을 순회하면서 테이블 안에 이름에 해당하는 값이 있을 경우 등장 횟수를 - 1일 하고, 등장 횟수가 0 이거나 name에 해당하는 property가 없는 경우 해당 name이 정답이 되게 된다.
```

### 느낀점

```markdown
이중 for 문을 사용하니 계속 시간 초과 오류가 발생했는데, 다른 사람 풀이에 있는 해시 테이블을 이용한 풀이를 사용해서 풀어보았다.
```
